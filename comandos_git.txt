git status : Revisar el estatus de los items que están en git
git add [ruta/name_of_file.ext] : Añadir un nuevo item a git para luego hacer commit | Tambien se puede aplicar a carpetas completas
git commit [name_of_file.ext] -m "Comentario breve" : Hacer commit a los items que han sido añadidos | Si no se especifica el archivo, se hace commit a todo
git push : "Empujar" los cambios a github
git pull : Mandar a traer los cambios de github a nuestro local
